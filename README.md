# Vibe Coding Class

An online, weekly vibe coding class for students in grades 6–10: 13 classes plus a separate Demo Day, 60 minutes each.

Students learn to build and ship real, deployed products (webpages, quizzes, tools, small games) by prompting AI tools like Lovable and Replit rather than writing code by hand — with an emphasis on prompt precision, iteration, evaluating AI output, and basic data ethics. The class culminates in a capstone product that each student presents live on Demo Day.

## Structure

- **Format:** Weekly, online, 60 minutes per class
- **Length:** 13 classes + 1 Demo Day (scheduled about a week after Class 13; Demo Day itself runs longer, scaling with roster size)
- **Audience:** Grades 6–10, with grade-band differentiation built into the domain build and capstone phases

| Phase | Classes | Focus |
|-------|---------|-------|
| Orientation and First Wins | 1–2 | Tools, first output, what prompting is |
| Prompt Skills and Iteration | 3–5 | Anatomy, iteration, error recognition |
| Domain Builds | 6–11 | Independent product development |
| Capstone Prep | 12–13 | Documentation, rehearsal |
| Demo Day | separate day | Live presentations |
